Web TESS Viewing Tool updated for Cycle 2

Date Tue 04 December 2018 By Tom Barclay

The popular tool for determining when a target will be observed by TESS -- Web TESS Viewing Tool (WTV) -- has been updated to include both year 1 and year 2 of TESS.
